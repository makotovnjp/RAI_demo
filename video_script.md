>  RAIエージェントがClaude APIに問い合わせ、

**操作**: アーキテクチャ図の各層をハイライト

### [01:00 - 02:00] シナリオ1: 基本操作コマンド

**画面**: ターミナル + アーム側面カメラ
**操作手順**:
1. ターミナルで `python3 rai_lite6_agent.py` を実行
2. プロンプトに入力: `ホームポジションに移動してグリッパーを開いてください`
3. エージェントの思考ログが流れる様子を表示
4. アームがゆっくりホームへ動き、グリッパーが開く

**ナレーション**:
> 「エージェントがLLMに問い合わせ、適切なツールコールの順序を決定します。
>  ログを見ると move_home → control_gripper(1.0) の順で実行されていますね」

**キャプチャするターミナル出力例**:
```
[You] ホームポジションに移動してグリッパーを開いてください

[Agent] タスクを理解しました。まずホームポジションに移動してから
        グリッパーを開きます。

[Tool→] move_home({})
[←Tool] ホームへ移動: 移動成功: (300.0, 0.0, 350.0) mm

[Tool→] control_gripper({"position": 1.0})
[←Tool] グリッパー開き(1.00): 成功

[Agent] ホームポジションへの移動とグリッパーオープンが完了しました。
```

---

### [02:00 - 03:30] シナリオ2: VLM物体認識 + ピック＆プレース

**画面**: 俯瞰カメラ + ターミナル（2分割）
**操作手順**:
1. 入力: `テーブルを確認して、赤いブロックをつかんで右側に移動してください`
2. `get_camera_image` が呼ばれ、VLMが画像を解析
3. 物体座標が推定され、アプローチ→把持→移送→解放のシーケンスが実行される

**ナレーション**:
> 「エージェントがまずカメラ画像を取得し、VLMで物体の位置を推定します。
>  次に、アプローチ高から降下して把持、持ち上げて移送という
>  一連の動作をLLMが計画・実行します」

**キャプチャするターミナル出力例**:
```
[You] テーブルを確認して、赤いブロックをつかんで右側に移動してください

[Tool→] get_camera_image({})
[←Tool] 画像取得: 1280x720px, 輝度=128.4, 保存先=capture_1747039200.jpg

[Agent] カメラ画像を確認しました。赤いブロックが中央やや左に見えます。
        把持に向けて移動します。

[Tool→] move_to_pose({"x": 300, "y": 100, "z": 250, "speed": 100})
[←Tool] 移動成功: (300.0, 100.0, 250.0) mm

[Tool→] control_gripper({"position": 1.0})
[←Tool] グリッパー開き(1.00): 成功

[Tool→] move_to_pose({"x": 300, "y": 100, "z": 80, "speed": 50})
[←Tool] 移動成功: (300.0, 100.0, 80.0) mm

[Tool→] control_gripper({"position": 0.2})
[←Tool] グリッパー閉じ(0.20): 成功
...（以下続く）
```

### [03:30 - 04:30] シナリオ3: マルチステップ自律タスク

**画面**: 全体俯瞰カメラ（複数ブロックが見える）
**操作手順**:
1. `python3 pick_and_place_demo.py` を実行（環境変数設定済み）
2. VLMが複数の物体を自動検出
3. 順番にピック＆プレースを実行
4. 完了後に実行ログが表示される

**ナレーション**:
> 「最後は完全自律モードです。事前に場所を教えなくても、
>  VLMがカメラで全物体を認識し、計画を立てて全て配置し直します」

## 録画後の編集メモ

| 編集項目 | 内容 |
|----------|------|
| BGM | ロイヤリティフリー（テック系）を低音量で |
| 字幕 | ナレーション全文 + ターミナル出力の重要部分 |
| ズーム | ターミナル文字が小さい場合は要拡大 |
| カット | ツール実行中の待ち時間（2秒以上）は1/4速またはカット |
| テロップ | 各シーン開始時に「シナリオX」のタイトルカード挿入 |
| サムネイル | アームがブロックをつかんでいる瞬間の写真 |
